##  使用


> 在启动此组件项目前，假设您已经安装了 scss 依赖，hx 版本 版本为3.2以上。启动此项目须先准备云服务空间。

直接在 hBuilder 中打开项目，关联云服务空间，启动即可。

>注：1、小程序和后台需使用同一个云服务空间 <br/>2、若遇到云函数依赖报错情况,请先尝试更新公共模块依赖
